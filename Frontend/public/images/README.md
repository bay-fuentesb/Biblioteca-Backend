# Carpeta de Imágenes

Descarga tus imágenes aquí. Las imágenes se pueden acceder en el frontend usando rutas como:
- `/images/nombre-imagen.jpg`
- `/images/nombre-imagen.png`
- `/images/nombre-imagen.webp`

## Ejemplo de uso en componentes:

```typescript
// En el template HTML
<img src="/images/mi-imagen.jpg" alt="Descripción">

// O en CSS
background-image: url('/images/mi-imagen.jpg');
```

Cuando tengas las imágenes descargadas, cuéntame:
1. Qué imágenes tienes
2. Dónde quieres usarlas en el proyecto
3. Cómo las quieres mostrar (tamaño, posición, etc.)
